# 🚀 EduFlow - Modern LMS Dashboard & Course Platform

A premium, fully responsive Learning Management System (LMS) frontend built with **React.js**, **Tailwind CSS**, and **Framer Motion**. This project features a stunning UI/UX designed for modern e-learning platforms like Udemy or Coursera.

![LMS Dashboard Preview](https://images.unsplash.com/photo-1501504905252-473c47e087f8?w=1200&auto=format&fit=crop&q=80)

## ✨ Features

- **📱 Fully Responsive Design**: Seamless experience across Desktop, Tablet, and Mobile devices using Tailwind's mobile-first approach.
- **🎨 Modern UI/UX**: Professional aesthetics featuring Glassmorphism, smooth Framer Motion animations, and custom scrollbars.
- **📊 Student Dashboard**: Comprehensive overview with learning stats, progress bars, and interactive activity charts.
- **📚 Dynamic Course Listings**: Interactive course cards with hover effects, categories, ratings, and instructor details.
- **🔐 Auth Pages**: Beautifully designed Student Login and Registration pages with social authentication options.
- **🧩 Modular Architecture**: Clean, reusable React component structure for easy scalability.

## 🛠️ Tech Stack

- **Framework**: [React.js](https://reactjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Routing**: [React Router Dom](https://reactrouter.com/)

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/YOUR_USERNAME/lmsproject.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm start
   ```

4. **Open in browser**
   Navigate to `http://localhost:3000`

## 📁 Project Structure

```text
src/
├── components/     # Reusable UI components (Navbar, Hero, CourseCard, etc.)
├── pages/          # Main application pages (Home, Dashboard, Login, Register)
├── index.css       # Global styles and Tailwind directives
└── App.js          # Main application routing
```
